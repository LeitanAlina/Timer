Простой веб-таймер обратного отсчета, созданный с использованием HTML, CSS и JavaScript.

Возможности: Ввод времени пользователем (от 1 до 99 минут), Запуск обратного отсчета по нажатию кнопки «Старт», Отображение времени в формате мм:сс, Круговой SVG-прогресс-бар с синхронной анимацией, Спиннер во время работы таймера, Возможность повторного запуска после завершения, Адаптивный пользовательский интерфейс.
